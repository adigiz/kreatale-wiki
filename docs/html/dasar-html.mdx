# Dasar HTML

:::tip

Selalu berikan **closing tag** saat menulis sebuah elemen html.

:::

## Formatting

HTML Formatting adalah proses untuk memformat teks agar memiliki _look and feel_ yang lebih baik. Element-element formatting didesain untuk menampilkan teks-teks spesial tanpa menggunakan CSS.

| Tag        | Kegunaan                  |
| ---------- | ------------------------- |
| `<strong>` | Menebalkan teks penting   |
| `<b>`      | Menebalkan teks           |
| `<em>`     | Menegaskan teks           |
| `<i>`      | Memiringkan teks          |
| `<u>`      | Menggaris bawahi teks     |
| `<mark>`   | Menyoroti teks            |
| `<sub>`    | Subscript teks            |
| `<sup>`    | Superscript teks          |
| `<big>`    | Membesarkan font 1 ukuran |
| `<small>`  | Teks kecil                |
| `<del>`    | Mencoret teks             |

## Headings

- Heading di tandai dengan tag `<h1>` hingga `<h6>`. Semakin kecilnya angka, menunjukkan bahwa semakin pentinglah heading tersebut.

## Paragraph

- HTML element `<p>` merepresentasikan sebuah paragraf
- Paragraphs are block-level elements (always goes to next line), and will automatically close if another block-level element is parsed before the closing `</p>` tag.
- Paragraphs are usually represented in visual media as blocks of text separated from adjacent blocks by blank lines and/or the first-line indentation

## Image
Ada banyak format foto / gambar yang bisa digunakan di web antara lain:
- BMP (Bitmap)
- GIF (Graphics Interchange Format)
- JPEG / JPG (Joint Photographic Experts Group)
- PNG (Portable Network Graphic)
- SVG (Scalable Vector Graphics)
- WebP (Web Picture format)

Gambar adalah salah satu hal yang paling penting dan paling dibutuhkan untuk membuat sebuah halaman web menjadi lebih menarik dan interaktif
- Di HTML, gambar di definisikan menggunakan tag `<img>`
- Pada tag tersebut, attribute `src` wajib di tambahkan dan attribute tersebut harus berisi URL dari gambar yang ingin di tampilkan
- Attribute `alt` digunakan untuk mendeskripsikan secara teks, apa gambar yang kita tampilkan. Attribute ini tidak wajib untuk ditambahkan namun akan sangat meningkatkan accessibility pengguna
